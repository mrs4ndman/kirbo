# Kirbo: my own spin on colorschemes

---

## Installation:

- `lazy.nvim`
```lua
return {
    "mrs4ndman/kirbo",
    enabled = true,
    lazy = false,
    config = function() 
      vim.cmd("colorscheme kirbo")
    end
}
```
